# project-ionic4-tool-multiedit

## Usage:

### mgit-init
Execute the command
```
cd project-ionic4-tool-multiedit
chmod +x mgit-init
./mgit-init <github-user-name>
```

---

### mgit-clone
Please execute the mgit-init first and make a repositry list.

Next, Execute the command
```
cd project-ionic4-tool-multiedit
chmod +x mgit-clone
./mgit-clone
```
